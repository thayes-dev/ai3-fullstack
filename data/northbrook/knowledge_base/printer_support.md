# Printer Support — Troubleshooting & Escalation

## Self-Service Troubleshooting

### Paper Jams
1. Open the front panel of the printer
2. Gently pull jammed paper in the direction of the paper path (do NOT pull backwards)
3. Check the rear access panel for additional jammed sheets
4. Close all panels firmly until they click
5. Press the "Resume" button or power cycle the printer

### Print Quality Issues
- **Streaks or fading:** Open the toner access panel and gently shake the toner cartridge side to side. Reinstall and try again.
- **Smudging:** Run a cleaning cycle from the printer's control panel (Settings → Maintenance → Clean).
- **Wrong colors:** Ensure you're printing with the correct color profile (Settings → Print Quality → "Standard").

### Connection Issues
- Confirm you're connected to the `NB-Internal` WiFi network (printers are not accessible from guest WiFi)
- Re-add the printer: System Preferences → Printers → "+" → search for the printer name (format: `NB-FL[floor]-[location]`, e.g., `NB-FL2-EAST`)

## Escalation
If self-service steps don't resolve the issue, submit an IT ticket with:
- Printer name (e.g., NB-FL2-EAST)
- Error message or symptoms
- How many times the issue has occurred

Recurring issues (3+ jams per week) are escalated to the vendor for service or replacement. **SLA:** Printer hardware issues — 2 business day resolution.
